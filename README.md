Интернет-магазин светильников с каталогом товаров, фильтрацией, сортировкой, корзиной и адаптивным интерфейсом.
Реализован на чистом JavaScript с интеграцией внешних библиотек.

Функциональность:
1)Навигация и интерфейс:
-Бургерное меню каталога — открытие/закрытие при клике на иконку меню;
-Выбор города — выпадающий список с возможностью смены локации, данные о наличии товаров в каждом городе отображаются в подсказках;

2)Каталог товаров:
-Динамическая отрисовка карточек — товары загружаются из файла data.json и отображаются на странице;
-Фильтрация — по типу светильников и наличию на складе (с учетом выбранного города);
-Сортировка — по цене (дорогие/дешевые) и популярности (рейтинг);
-Пагинация — разбивка каталога на страницы (по 6 товаров на странице) при большом количестве товаров;
-Подсчет количества товаров — рядом с каждым фильтром отображается актуальное количество товаров в категории;

3)Корзина:
-Добавление товаров — при нажатии на кнопку «В корзину» товар добавляется в корзину, обновляется счетчик;
-Удаление товаров — возможность удалить отдельный товар из корзины;
-Отображение корзины — выпадающее меню с перечнем добавленных товаров;
-Пустая корзина — отображается специальный блок, если корзина пуста;

4)Товары дня (слайдер):
-Swiper.js — интерактивный слайдер с товарами, у которых установлен флаг goodsOfDay: true;
-Навигация — стрелки для переключения слайдов;
-Адаптивность — настройка количества видимых слайдов в зависимости от ширины экрана;

5)Аккордеон (FAQ):
-Вопросы и ответы — блок с часто задаваемыми вопросами в формате аккордеона;
-Одновременное открытие — только один пункт может быть открыт в каждый момент времени;
-Закрытие по клику — повторный клик по активному вопросу закрывает ответ;

6)Форма обратной связи:
-Валидация — реализована с помощью библиотеки Just Validate;
-Имя: обязательно, от 3 до 20 символов;
-Email: обязательно, валидация по формату email;
-Чекбокс согласия: обязателен к выбору;
-Отправка на сервер — POST-запрос на https://httpbin.org/post;
-Модальные окна — уведомления об успешной отправке или ошибке с возможностью закрытия;

7)Всплывающие подсказки (тултипы):
-Tippy.js — при наведении на иконку i на карточке товара появляется подсказка;
-Информация о наличии — отображается доступность товара в каждом городе (Москва, Санкт-Петербург, Оренбург);
